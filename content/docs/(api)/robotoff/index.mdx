# Robotoff API

Robotoff is Open Food Facts' machine learning service that provides AI-powered predictions for product data and enables crowd-sourced validation through annotations. It helps improve data quality by suggesting corrections and enhancements.

<Callout type='info'>
  Robotoff provides both production (`robotoff.openfoodfacts.org`) and staging
  (`robotoff.openfoodfacts.net`) environments for testing and development.
</Callout>

## Base URL

All endpoints are prefixed with `/api/v1`:

```
https://robotoff.openfoodfacts.org/api/v1/{endpoint}
```

## Authentication

Most read operations are public. For write operations (annotations, batch processing), you may need to provide Open Food Facts credentials using Basic Authentication.

<Callout type="warn">
When submitting annotations as a registered user, include `Authorization: Basic {base64(username:password)}` header.
</Callout>

## Prediction Types

Robotoff provides predictions for:

- **Categories**: Product category classification
- **Ingredients**: Ingredient list parsing and analysis
- **Nutrition**: Nutrition facts extraction from images
- **Labels**: Detection of quality labels and certifications
- **Brands**: Brand identification and verification
- **Packaging**: Package type and material detection

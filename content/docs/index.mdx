# Open Food Facts API Documentation

Welcome to the comprehensive API documentation for the Open Food Facts ecosystem. Our APIs provide access to the world's largest collaborative food database and related services.

<Callout type='info'>
  Open Food Facts is a collaborative, free and open database of food products from around the world.
  All data is available under open licenses.
</Callout>

## Getting Started

Choose the API that best fits your needs:

- **Product Data**: Use API v2 or v3 for accessing product information, nutrition data, and images
- **Smart Insights**: Use Knowledge Panels for structured, ready-to-display product information
- **AI Predictions**: Use Robotoff for machine learning predictions and annotation workflows
- **Price Information**: Use Open Prices for product pricing data and store locations

<Callout type='success'>
  All APIs are free to use and don't require authentication for read operations. Check individual
  API sections for specific usage guidelines.
</Callout>

# Open Food Facts API v2 Documentation

This section contains documentation for the original Open Food Facts API (version 2). 

## Available Endpoints

The API is organized by the following categories:

- **Read Requests**: Endpoints for retrieving product information, searching products, and more
- **Write Requests**: Endpoints for uploading, updating product information and images
- **Authentication**: Endpoints for session management and authentication
- **Personal Search**: Endpoints for personalized product searches and preferences

## Getting Started

To get started, navigate to one of the endpoint categories in the sidebar. The most commonly used endpoint is [Get Product by Barcode](/docs/(api)/v2/read--requests/get-product-by-barcode) which allows you to retrieve detailed product information using a barcode.

## API Base URL

The base URL for all v2 API endpoints is:

```
https://world.openfoodfacts.org
```

For more information about Open Food Facts data, visit our [data documentation](https://world.openfoodfacts.org/data).
